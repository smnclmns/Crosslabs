Welcome to the Interactive Titration Simulation Tutorial!

Project Overview

Our project aims to provide students with a deep understanding of automating a titration. Through an Arduino-based hardware solution, a user-friendly web application, and interactive pseudo-code tasks, students can learn the individual steps of titration and its automation.

Scientific Background

Titration is a fundamental procedure in chemistry used to determine the concentration of a known reactant in a solution. By automating this process, we can improve accuracy, repeatability, and efficiency.

Our Technology

We use an Arduino microcontroller to automate the titration step. In addition, we use Flask as a web framework to create a user-friendly interface through which users can understand and modify the Arduino code and its functions.

How It Works

In this interactive tutorial, students learn how the automation of titration works by completing pseudo-code and matching it with the corresponding functions of the Arduino code. Students' progress is evaluated through a scoring system, and helpful tips support them in improving their skills.

Get Started

To get started, visit the Workspace and start the tutorial. Enjoy learning!
